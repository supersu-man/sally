## Installation & Setup

1. Clone the repository
```
git clone https://github.com/supersu-man/sally.git
cd sally
```

2. Install dependencies
```
npm install
```
3. Run the development server
```
ng serve
```
4. Build for production
```
ng build --configuration production
```
